# Smart Traffic Management System

An AI-based Smart Traffic Management System using YOLOv8 for real-time vehicle detection and adaptive traffic signal optimization.

## Features

- Vehicle Detection using YOLOv8
- Dynamic Green Signal Timing
- Lane-wise Vehicle Counting
- ROI Selection
- Flask Backend
- Responsive Frontend
- Future Support for Ambulance Priority

## Technologies

- Python
- Flask
- OpenCV
- YOLOv8
- HTML
- CSS
- JavaScript
